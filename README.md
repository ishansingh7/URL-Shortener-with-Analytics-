# 🔗 Smart URL Shortener

A modern and professional **MERN Stack URL Shortener Application** that allows users to generate short URLs instantly, create QR codes, track clicks, and manage all URLs from a powerful analytics dashboard.

Built with a premium UI/UX design using **React.js**, **Node.js**, **Express.js**, and **MongoDB**.

# 🚀 Features

## 🔐 Authentication System
- User Registration & Login

- Protected Routes

## 🔗 URL Shortening
- Convert long URLs into short links instantly
- Fast redirect system
- Unique short code generation

## 📱 QR Code Generation
- Auto-generate QR codes for every short URL

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create `.env` file inside backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
BASE_URL=http://localhost:5000
```
